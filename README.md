## Overview

This project is an Information Retrieval system built for the CSA2001 - Fundamentals in AI & ML course. It acts as a search engine that allows users to query the **20 Newsgroups dataset**. By utilizing Term Frequency-Inverse Document Frequency (TF-IDF) and Cosine Similarity, the system accurately retrieves and ranks the most relevant text documents based on user input.

## Features

* **Text Vectorization:** Converts unstructured text into numerical data using TF-IDF.

* **Fast Querying:** Uses pre-computed, serialized sparse matrices to deliver search results almost instantly.

* **Cosine Similarity Matching:** Ranks documents based on the angle between the query vector and document vectors, ensuring accurate contextual matches.

## Repository Structure

* `AIML/dataset/20news-bydate_py3.pkz`: The compressed 20 Newsgroups dataset used for the corpus.

* `AIML/IDF-matrix_updated.py`: The preprocessing script. It reads the dataset, fits the TF-IDF vectorizer, transforms the text, and exports the models.

* `AIML/IDF-search_updated.py`: The main search application script. It loads the saved models and processes user queries to find matching documents.

* `AIML/*.joblib`: Serialized model files (Vectorizer, Matrix, Target Names) generated by the matrix script to ensure fast search execution without retraining.

## Setup and Installation

### Prerequisites

Make sure you have Python installed on your system along with the following libraries:

* `scikit-learn`

* `numpy`

* `scipy`

* `joblib`

You can install the dependencies using pip:
`pip install scikit-learn numpy scipy joblib`

## How to Run

**Step 1: Generate the TF-IDF Matrix (Optional if `.joblib` files are present)**
If you are running this for the first time or want to rebuild the model from the dataset, run the matrix generation script. *Note: This might take a minute as it processes thousands of text files.*
`python AIML/IDF-matrix_updated.py`
This will generate the `_verified.joblib` and `_aligned.joblib` files in your directory.

**Step 2: Run the Search Engine**
To start querying the dataset, run the search script:
`python AIML/IDF-search_updated.py`

When the prompt appears, type your search query (e.g., "space exploration", "computer hardware", or "encryption"). The script will return the top matching documents from the 20 Newsgroups dataset, ranked by relevance.
